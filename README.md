## Tentang

Lua Obfuscator by Paanz Studio adalah alat berbasis web untuk melindungi kode Lua agar tidak bisa dibaca, di-decompile, atau dimodifikasi oleh pihak yang tidak berwenang. Kode yang sudah diobfuscasi tetap berjalan **100% sama** seperti aslinya namun sudah diacak melalui 7 lapisan proteksi secara bersamaan.

Seluruh proses berjalan di browser. **Tidak ada kode yang dikirim ke server manapun.**

---

## Fitur

| Fitur | Deskripsi |
|:------|:----------|
| 🛡️ 7-Layer Obfuscation | UTF-8 encoding, shift cipher, hex encoding, chunk splitting, chunk reordering, variable mangling, dead code injection |
| 📁 Sistem Folder | Kelola banyak kode Lua dalam folder yang terorganisir |
| 💾 Auto-Save Permanen | Data tersimpan di perangkat user, tidak hilang meskipun browser ditutup |
| 👁️ Dua Tampilan | Tab terpisah untuk melihat kode asli dan kode terobfuscasi |
| 📋 One-Click Copy | Salin kode asli maupun hasil obfuscasi langsung ke clipboard |
| 📱 Responsif | Berfungsi optimal di desktop, tablet, dan smartphone |
| ✒️ Watermark Otomatis | Setiap kode terobfuscasi menyertakan watermark tanpa mengganggu fungsi kode |
| ⌨️ Keyboard Shortcut | Ctrl+S simpan, Ctrl+Shift+O obfuscasi, Tab indent, Escape tutup modal |

## Cara Menggunakan

<div align="center">

```
Buka Web → Buat Folder → Buat Kode → Tempel Kode → Obfuscasi → Salin Hasil
```

</div>

**1.** Buka [Lua Obfuscator by Paanz Studio](https://alfanrizkyw.github.io/Lua-Obfuscator-By-Paanz-Studio/)

**2.** Klik **Folder Baru** lalu beri nama folder

**3.** Klik ikon **+** pada folder, beri nama kode

**4.** Tempelkan kode Lua kamu di tab **Kode Asli**

**5.** Klik tombol **Obfuscasi**

**6.** Pindah ke tab **Terobfuscasi** lalu klik **Salin Hasil**

## Lapisan Proteksi

```
KODE LUA ASLI
       │
       ▼
  [1] UTF-8 Encoding        → Konversi teks ke byte UTF-8
       │
       ▼
  [2] Shift Cipher           → Geser setiap byte dengan kunci acak (1-200)
       │
       ▼
  [3] Hex Encoding           → Konversi ke representasi heksadesimal
       │
       ▼
  [4] Chunk Splitting        → Pecah data menjadi 2-6 bagian terpisah
       │
       ▼
  [5] Chunk Reordering       → Acak urutan bagian secara acak
       │
       ▼
  [6] Variable Mangling      → Ganti semua nama variabel jadi string acak 7-14 karakter
       │
       ▼
  [7] Dead Code Injection    → Sisipkan kode palsu antara baris-baris penting
       │
       ▼
KODE TEROBFUSKASI (tidak bisa dibaca, tetap berjalan 100%)
```
