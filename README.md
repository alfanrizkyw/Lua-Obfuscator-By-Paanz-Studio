***Lua Obfuscator by Paanz Studio***

Alat online untuk melindungi kode Lua agar tidak bisa dibaca atau di-decompile. Kode diobfuscasi melalui 7 lapisan proteksi dan tetap berjalan 100% seperti aslinya. Semua proses di browser, tidak ada data yang dikirim ke server.

***Cara Penggunaan***

Buka web
Buat folder baru
Buat kode baru di dalam folder
Tempelkan kode Lua di tab Kode Asli
Klik tombol Obfuscasi
Salin hasilnya di tab Terobfuscasi
